Data sets used on this website

  • All data available from the Registry of Open Access Repository Mandates and Policies (ROARMAP)
  • Bibliographic metadata indexed by the Thompson-Reuters database (WoK)
  • Data from the Altmetrics database

Other Open Access related data sets

  • The Directory of Open Access Repositories - OpenDOAR - data can be extracted using RSS or screen scraping.
